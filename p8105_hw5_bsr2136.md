### Function to remove missing values

``` r
remove_missing <- function(x) {
  
  if (is.numeric(x)) {
      for (i in 1:length(x)) {
          if (is.na(x[i])) {
              x[i] = mean(x,na.rm=T)
          }
      }
  }
  else if (is.character(x)) {
          for (i in 1:length(x)) {
              if (is.na(x[i])) {
                  x[i] = "virginica"
              }
          }
  }
  else stop("Not numeric or character")
  x
}
```
